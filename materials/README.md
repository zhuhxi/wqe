# Course Materials Structure

This directory organizes all course files by week/theme.

## Layout

- `weekXX_.../pdf/`:
  Original PDF slides for that week/theme.
- `weekXX_.../text/`:
  Text-extracted version of the same slides.

## Week Mapping

- `week01_introduction`
- `week02_foundations`
- `week03-04_authentication`
- `week05-06_access-control`
- `week07_malware`
- `week08-09_network-security-foundations`
- `week10-11_security-protocols`
- `week12_network-design-hardening`
- `week13-14_firewall-vpn-ids`

## Notes

- Temporary/intermediate files (e.g. `*.aux`, `*.log`, `*.out`, `*.toc`) are removed.
- Legacy flat directories (`cybersecurity-ppts/`, `review-text/`) were consolidated into this structure.
